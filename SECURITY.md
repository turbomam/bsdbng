# Security Policy

## Reporting a Vulnerability

Open a private security advisory if the feature is available, or contact the
maintainer directly if you need to avoid public disclosure while GitHub-side
security features are being finalized.

## Current Protections

- CodeQL
- Dependabot version updates
- Detect-secrets in pre-commit
- `pip-audit` in local and CI quality gates
- `bandit` for Python security checks
